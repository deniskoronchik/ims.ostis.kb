Задачей агента проверки наличия циклов в отношении порядка в рамках заданной структуры является поиск цикла для всех отношений порядка, которые принадлежат заданной структуре. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу проверки наличия циклов в любом отношении порядка. В данном агенте предусмотренна возможность поиска циклов у всех отношений порядка, который предусмотренны в базе знаний на данный момент. Для этого на вход агент стоит подать структуру, которой будет принадлежать узел отношение порядка. В заданной структуре может находится неограниченное количество отношений порядка, но в рамках каждого отдельно взятого отношения агент может выявить только один цикл. При необходимости ограничения области поиска агента в качестве второго аргумента запроса подается контекст действия, в рамках которого будет осуществляться поиск.
